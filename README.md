## 📚 Conteúdo do Curso
- Introdução ao HTML5
- Estrutura básica de páginas
- Trabalhando com textos e links
- Inserindo imagens, áudios e vídeos
- Formatação com CSS3
- Cores, fontes e seletores
- Modelos de caixas (box model)
- Layouts e responsividade
